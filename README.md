### Projetos em Destaque

* **Trabalho de Conclusão de Curso (TCC):** Desenvolvimento de uma pipeline de Visão Computacional para reconhecimento de jogadas e verificação de **Expected Goals (xG)** em partidas de futebol. Utilização de **YOLO**, **OpenCV** e o dataset **SoccerNet** para análise preditiva em vídeo.
* **Sistemas Especialistas e Simulações:** Implementação de motores de inferência para auxílio em diagnósticos e desenvolvimento de simulações físicas 3D utilizando as engines Ursina e Panda3D em Python.
* **Desenvolvimento de Software:** Criação de aplicações escaláveis utilizando **Flutter** com integração ao Firebase e estruturação de sistemas web com **Laravel**.
